Projeto de Férias

Simulador de Batalha com Lógica Avançada

Revisão Rápida: Conceitos Essenciais

Olá, Programador! Para esta versão final do projeto, vamos integrar todas as ferramentas que aprendemos, incluindo a capacidade de tomar decisões complexas.

  • Estrutura Básica, Variáveis e Constantes: O uso de inteiro, real, cadeia, logico e const continua sendo a base de tudo.
  • Entrada e Saída: Os comandos escreva() e leia() serão usados para tornar o programa interativo.
  • Operadores Aritméticos: Você precisará de +, -, * e / para os cálculos de dano.
  • Operadores Relacionais e Lógicos: Esta é a parte crucial! Você combinará operadores como "maior ou igual a" com os operadores lógicos e e ou para criar as regras do jogo.
  • Estruturas Condicionais: Use se, senao se e senao para que o programa execute ações diferentes com base nas regras que você criar.

Visão Geral e Enunciado do Projeto (v3.0)

Visão Geral: Seu desafio é criar um simulador de rodada de combate que verifique requisitos de personagem mais complexos. O programa decidirá se um ataque pode ser feito e, se puder, calculará o dano e o resultado da batalha com base em regras diferentes.

⚠️ Importante: Lembre-se de criar **seus próprios nomes para as variáveis**. O objetivo é que você crie um código que **você** entenda facilmente.

Passo a Passo do Projeto:

  1. Parte 1: Definindo os Atributos e Requisitos

    Defina as constantes do Monstro (Nome: "Cyclops", Vida: 260, Fator de Defesa: 1.1) e as constantes para os requisitos da arma (Level Normal: 20, Skill Normal: 30, Magic Level de Mago: 15). Depois, declare todas as variáveis que você precisará para os dados do Herói e para os cálculos.

    // Constantes do Monstro
const cadeia NOME_MONSTRO = "Cyclops"
const inteiro VIDA_INICIAL_MONSTRO = 260
const real FATOR_DEFESA_MONSTRO = 1.1

// Constantes de Requisitos
const inteiro REQUISITO_LEVEL_NORMAL = 20
const inteiro REQUISITO_SKILL_NORMAL = 30
const inteiro REQUISITO_ML_MAGO = 15

// Variáveis do Herói e de Controle
cadeia nomeDoPersonagem
inteiro levelDoPersonagem, skillDeAtaque, magicLevelDoPersonagem
caracter vocacaoDoPersonagem // 'K', 'P', 'M'
real danoBaseDaArma
logico podeEquiparArma
  2. Parte 2: Coletando os Dados do Herói

    Peça ao usuário para digitar o nome, level, skill de ataque, vocação (um único caracter: 'K' para Knight, 'P' para Paladin, 'M' para Mago), magic level e o dano base da arma.

    escreva("--- Simulador de Batalha v3.0: Lógica Avançada ---\n\n")
escreva("Digite o nome do seu Herói: ")
leia(nomeDoPersonagem)

escreva("Digite o level de ", nomeDoPersonagem, ": ")
leia(levelDoPersonagem)

escreva("Digite a skill de ataque (ex: Sword) de ", nomeDoPersonagem, ": ")
leia(skillDeAtaque)

escreva("Digite sua vocação ('K'night, 'P'aladin, 'M'ago): ")
leia(vocacaoDoPersonagem)

escreva("Digite seu Magic Level: ")
leia(magicLevelDoPersonagem)

escreva("Digite o dano base da sua arma: ")
leia(danoBaseDaArma)
  3. Parte 3: Verificação de Requisitos com Lógica Complexa

    A regra para usar a arma é: o herói precisa atender aos requisitos normais (level e skill) **OU** ser um Mago com o magic level mínimo. Crie uma expressão lógica que verifique esta condição e use uma estrutura `se...senao` para controlar o fluxo do programa.

    logico atendeRequisitoNormal = (levelDoPersonagem >= REQUISITO_LEVEL_NORMAL e skillDeAtaque >= REQUISITO_SKILL_NORMAL)
logico atendeRequisitoMago = (vocacaoDoPersonagem == 'M' e magicLevelDoPersonagem > REQUISITO_ML_MAGO)

podeEquiparArma = (atendeRequisitoNormal ou atendeRequisitoMago)

se (podeEquiparArma) {
    // O código das próximas etapas vai aqui...
} senao {
    escreva("\n--- ATAQUE FALHOU! ---\n")
    escreva("Você não atende aos requisitos para usar esta arma.\n")
}
  4. Parte 4: Calculando o Dano (Dentro do `se` principal)

    Se o herói puder equipar a arma, calcule o dano. Verifique se o dano é crítico (maior que 100) e, se for, aplique um bônus de 20%.

    escreva("\n--> Requisitos atendidos! Iniciando ataque...\n")

real danoBrutoDoHeroi = danoBaseDaArma + (skillDeAtaque * 0.5) + (levelDoPersonagem * 0.2)
			
se (danoBrutoDoHeroi > 100.0) {
    escreva("--> GOLPE CRÍTICO! Dano aumentado em 20%!\n")
    danoBrutoDoHeroi = danoBrutoDoHeroi * 1.20
}

real danoFinalNoMonstro = danoBrutoDoHeroi / FATOR_DEFESA_MONSTRO
real vidaAtualizadaDoMonstro = VIDA_INICIAL_MONSTRO - danoFinalNoMonstro
  5. Parte 5: Exibindo o Relatório da Rodada (Dentro do `se` principal)

    Mostre um relatório com os cálculos do ataque e use `se...senao` para exibir se o monstro foi derrotado ou se sobreviveu.

    escreva("\n--- RELATÓRIO DA RODADA DE COMBATE ---\n")
escreva("Dano Bruto do Ataque (com bônus, se houver): ", danoBrutoDoHeroi, "\n")
escreva("Dano final no ", NOME_MONSTRO, " (após defesa): ", danoFinalNoMonstro, "\n")
escreva("--------------------------------------------------\n")

se (vidaAtualizadaDoMonstro <= 0.0) {
    escreva("RESULTADO: O ", NOME_MONSTRO, " foi derrotado!\n")
} senao {
    escreva("RESULTADO: O ", NOME_MONSTRO, " sobreviveu com ", vidaAtualizadaDoMonstro, " de vida.\n")
}
escreva("--------------------------------------------------\n")

Gabarito (Solução Completa)

Tente resolver o projeto sozinho antes de consultar a solução! O gabarito é uma ferramenta de apoio para tirar dúvidas ou comparar sua lógica.


programa
{
	// --- Parte 1: Definindo os Atributos e Requisitos ---
	const cadeia NOME_MONSTRO = "Cyclops"
	const inteiro VIDA_INICIAL_MONSTRO = 260
	const real FATOR_DEFESA_MONSTRO = 1.1
	const inteiro REQUISITO_LEVEL_NORMAL = 20
	const inteiro REQUISITO_SKILL_NORMAL = 30
	const inteiro REQUISITO_ML_MAGO = 15

	cadeia nomeDoPersonagem
	inteiro levelDoPersonagem, skillDeAtaque, magicLevelDoPersonagem
	caracter vocacaoDoPersonagem // 'K', 'P', 'M'
	real danoBaseDaArma
	logico podeEquiparArma

	funcao inicio()
	{
		// --- Parte 2: Coletando os Dados do Herói ---
		escreva("--- Simulador de Batalha v3.0: Lógica Avançada ---\n\n")
		escreva("Digite o nome do seu Herói: ")
		leia(nomeDoPersonagem)
		
		escreva("Digite o level de ", nomeDoPersonagem, ": ")
		leia(levelDoPersonagem)
		
		escreva("Digite a skill de ataque (ex: Sword) de ", nomeDoPersonagem, ": ")
		leia(skillDeAtaque)

		escreva("Digite sua vocação ('K'night, 'P'aladin, 'M'ago): ")
		leia(vocacaoDoPersonagem)

		escreva("Digite seu Magic Level: ")
		leia(magicLevelDoPersonagem)
		
		escreva("Digite o dano base da sua arma: ")
		leia(danoBaseDaArma)

		// --- Parte 3: Verificação de Requisitos com Lógica Complexa ---
		logico atendeRequisitoNormal = (levelDoPersonagem >= REQUISITO_LEVEL_NORMAL e skillDeAtaque >= REQUISITO_SKILL_NORMAL)
		logico atendeRequisitoMago = (vocacaoDoPersonagem == 'M' e magicLevelDoPersonagem > REQUISITO_ML_MAGO)

		podeEquiparArma = (atendeRequisitoNormal ou atendeRequisitoMago)

		se (podeEquiparArma) {
			// --- Parte 4: Calculando o Dano ---
			escreva("\n--> Requisitos atendidos! Iniciando ataque...\n")

			real danoBrutoDoHeroi = danoBaseDaArma + (skillDeAtaque * 0.5) + (levelDoPersonagem * 0.2)
			
			se (danoBrutoDoHeroi > 100.0) {
				escreva("--> GOLPE CRÍTICO! Dano aumentado em 20%!\n")
				danoBrutoDoHeroi = danoBrutoDoHeroi * 1.20
			}

			real danoFinalNoMonstro = danoBrutoDoHeroi / FATOR_DEFESA_MONSTRO
			real vidaAtualizadaDoMonstro = VIDA_INICIAL_MONSTRO - danoFinalNoMonstro

			// --- Parte 5: Exibindo o Relatório ---
			escreva("\n--- RELATÓRIO DA RODADA DE COMBATE ---\n")
			escreva("Dano Bruto do Ataque (com bônus, se houver): ", danoBrutoDoHeroi, "\n")
			escreva("Dano final no ", NOME_MONSTRO, " (após defesa): ", danoFinalNoMonstro, "\n")
			escreva("--------------------------------------------------\n")

			se (vidaAtualizadaDoMonstro <= 0.0) {
				escreva("RESULTADO: O ", NOME_MONSTRO, " foi derrotado!\n")
			} senao {
				escreva("RESULTADO: O ", NOME_MONSTRO, " sobreviveu com ", vidaAtualizadaDoMonstro, " de vida.\n")
			}
			escreva("--------------------------------------------------\n")

		} senao {
			escreva("\n--- ATAQUE FALHOU! ---\n")
			escreva("Você não atende aos requisitos para usar esta arma.\n")
			escreva("(Requisito Normal: Lvl ", REQUISITO_LEVEL_NORMAL, " e Skill ", REQUISITO_SKILL_NORMAL, ")\n")
			escreva("(Requisito Alternativo: Ser Mago com ML > ", REQUISITO_ML_MAGO, ")\n")
		}
	}
}
