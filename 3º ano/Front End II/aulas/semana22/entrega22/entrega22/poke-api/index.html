PokeDex

Imagine que esta page é um lugar onde você irá mostrar os cards de Pokemon, como se fosse uma Pokedex, e você precisa puxar as informações e montar um time. Para isto você irá acessar o site abaixo, e utilizar o endpoint de busca de Pokemons.

API Base: PokeAPI

Instruções

  • O código deve tentar buscar 4 pokemons (por nome) através de um array, e exibi-lo no console.
  • Caso algum pokemon não seja encontrado, o código deve exibir uma mensagem de erro no console, mas continuar buscando os demais pokemons.
  • Ao final, o código deve exibir no console um array com os pokemons encontrados, e ignorar os que não foram encontrados.

Passo a Passo

  1. Crie uma função assíncrona que irá buscar os pokemons através do endpoint da PokeAPI, onde:
    • Cria a função assincrona e recebe o nome do pokemon como parâmetro;
    • Faz uma requisição HTTP (link da url) para o endpoint da PokeAPI;
    • Verifica o status da resposta e lança um erro caso o pokemon não seja encontrado;
    • Retorna os dados do pokemon em caso de sucesso;
      •  
    


// Cria a função assíncrona e recebe o nome do pokemon como parâmetro
async function buscarPokemon(nome) {    /*Faz uma requisição HTTP para o endpoint da PokeAPI.
    O await aguarda a Promise retornada pelo fetch ser resolvida. */
    try {
        const response = await fetch(`https://pokeapi.co/api/v2/pokemon/${nome}`);        // Verifica se a resposta HTTP foi bem-sucedida (com base no status da resposta do servidor.)
        if (!response.ok) {
            throw new Error(`Pokémon "${nome}" não foi encontrado no banco de dados.`);
        }        // Converte a resposta para JSON e retorna os dados do Pokémon:
        const pokemonConvertido = await response.json();
        return pokemonConvertido;
        }    // Captura e exibe o erro de forma controlada
    catch (error) {
        console.warn("Aviso de erro tratado:", error.message);

        // Retorna null para indicar falha sem interromper a execução
        return null;
    }
    }
  2. Crie um array com 4 nomes de pokemons, sendo que pelo menos 1 deles não deve existir, e utilize o método .map para chamar a função de busca para cada pokemon do array. Nota: 'agumon' é um digimon, e está sendo inserido para demonstrar o tratamento de erros. 
                        
        let timePokemon = ['pikachu', 'charizard', 'mewtwo', 'agumon'];
        let promisesPokemon = timePokemon.map(nome => buscarPokemon(nome));

  3. Utilize o Promise.all() para aguardar todas as promises serem resolvidas, e depois um console.log() para exibir o resultado final, filtrando os valores nulos. 
                        
        Promise.all(promisesPokemon).then(resultadoFinal => {
            // Filtra os nulls gerados pelos erros se quiserem uma lista 100% limpa
            const esquadraoValido = resultadoFinal.filter(p => p !== null);
            console.log("Esquadrão Pronto para os Web Components:", esquadraoValido);
        });
