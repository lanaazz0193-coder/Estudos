Streaming Characters

Imagine que esta página faz parte de uma plataforma de streaming e que você precisa gerar os cards dos personagens principais da série internacionalmente conhecida como Rick and Morty.

Para isso, você deverá consultar uma API pública e obter os dados dos personagens através de seus respectivos IDs.

API Base: Rick and Morty API

Instruções

  • Crie uma função assíncrona responsável por buscar os dados de um personagem da série Rick and Morty utilizando seu ID.
  • A função deve realizar a requisição para a API e tratar situações em que o personagem não seja encontrado.
  • Monte uma lista contendo 4 IDs de personagens, sendo que pelo menos um deles deve ser inválido para demonstrar o tratamento de erros.
  • Utilize o método .map() para disparar as buscas dos personagens.
  • Aguarde a conclusão de todas as requisições utilizando Promise.all().
  • Ao final, exiba no console apenas os personagens encontrados, ignorando os resultados inválidos.
  • O programa não deve interromper sua execução caso uma das buscas falhe.

Resultado Esperado

Ao finalizar a execução, o console deve apresentar um array contendo apenas os personagens válidos retornados pela API.
