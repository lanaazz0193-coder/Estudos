Disney+

Imagine que você está desenvolvendo uma plataforma de streaming e precisa gerar os cards dos personagens mais famosos da Disney.

Para isso, você deverá consultar uma API pública e obter informações sobre os personagens através de seus respectivos nomes.

API Base: https://api.disneyapi.dev/ Disney API

Instruções

  • Crie uma função assíncrona responsável por buscar os dados de um personagem da Disney utilizando seu nome.
  • A função deve realizar uma requisição para a API e tratar possíveis falhas durante a busca.
  • Monte uma lista contendo 4 nomes de personagens da Disney.
  • Pelo menos um dos nomes deve ser inválido para demonstrar o tratamento de erros da aplicação.
  • Utilize o método .map() para iniciar as buscas dos personagens presentes na lista.
  • Aguarde a conclusão de todas as requisições utilizando Promise.all().
  • Atenção: Analise que nesta atividade na API não retornará os dados do personagem. Na verdade, com esta busca você receberá a lista de (films) das quais cada personagem participou.
  • Ao final, exiba no console apenas os personagens encontrados com sucesso e suas respectivas listas de produções.
  • O programa deve continuar executando normalmente mesmo que uma ou mais buscas retornem erro.

Resultado Esperado

Ao término da execução, o console deverá apresentar um array contendo apenas os personagens válidos retornados pela API.

Cada item exibido deverá permitir a identificação do personagem e das produções da Disney nas quais ele participou.
