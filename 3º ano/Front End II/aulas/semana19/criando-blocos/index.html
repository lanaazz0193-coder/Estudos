Blocos Virtuais

🛠️ Missão: Galeria de blocos

Crie blocos virtuais na tela! A ação deve ocorrer quando o usuário clicar no botão ou apertar a tecla Enter.

  1. Capture o input, o botão e a <section> usando document.getElementById.
  2. Crie uma função para gerar o bloco com os seguintes passos:
    • Se o campo estiver vazio, mostre um alert() e pare a execução com return;
    • Crie uma nova <div> usando document.createElement;
    • Insira o nome digitado na div usando o textContent;
    • Injete o visual do Bootstrap usando o classList.add com as seguintes formatações: fundo verde, texto branco, bordas arredondadas e peso da fonte em negrito;
    • Faça o bloco aparecer na tela colocando-o na section com appendChild;
    • Limpe o campo de texto (value = '').
  3. Conecte essa função a dois ouvintes de eventos (addEventListener):
    • Um evento de 'click' no botão;
    • Um evento de 'keypress' no input.
💡 Dica: Para o evento de teclado, capture o evento na função (event) => { ... }, verifique se a tecla que foi pressionada é o Enter usando if (event.key == 'Enter'), e passe este 'event' para a função que cria o bloco.
