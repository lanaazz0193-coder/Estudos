Histórico do Sistema

Você precisa criar um sistema simples de log de ações. Toda vez que o usuário clicar no botão "Registrar Ação", a aplicação deve registrar a data e o horário exatos em que a ação ocorreu na tela.

Para isso, você precisa:

  • Usar o document.getElementById para capturar o botão de registrar e a <div> que servirá como container do histórico;
  • Adicionar um evento de clique (addEventListener) ao botão;
  • Dentro do evento de clique, criar as seguintes instruções DENTRO de uma arrow function (ação não nomeada):
    • Criar um novo elemento de parágrafo (<p>) com a ajuda do document.createElement;
    • Capturar o momento exato do clique instanciando a classe Date(), usando new Date();
    • Formatar essa data para o padrão brasileiro, para que ela fique no formato dd/mm/aaaa hh:mm:ss.
      💡 Dica: Vá até a documentação do W3Schools ou MDN Web Docs e pesquise sobre a classe Date e o método .toLocaleString() ou .toLocaleDateString(). Eles farão o trabalho pesado da formatação para você, e é uma habilidade que vocês PRECISAM aprender!
    • Inserir o texto "Ação registrada em data" dentro do parágrafo criado através do textContent;
    • Por fim, incluir esse novo parágrafo visualmente dentro da <div> do histórico utilizando o appendChild.
